# instaclone
Instagram feed clone with react native

## Tools
* React Native
* Typescript
* Styled Components
* Json-server

## Features
* Pull to refresh
* Infinite loading

## How to run 
* Run `yarn add` to install the dependencies
* Run `adb reverse tcp:3000 tcp:3000` to change your tcp port
* Run `yarn dev:server` to run the server
